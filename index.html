REST API
POST /leads — прямые запросы

Отправляйте лидов из любого сервиса через HTTP POST.

curl -X POST __BASE__/leads \
  -H "Content-Type: application/json" \
  -d '{
    "name":"Иван",
    "phone":"+79991234567",
    "pipeline_id":1,
    "client_type":"физ",
    "source":"Сайт — Заказать звонок",
    "telegram":"@ivan",
    "whatsapp":"+79991234567"
  }'

Swagger: /docs

Telegram-уведомления клинерам
Заявка «Выезд на объект» → уведомление

Когда у лида появляется статус «Выезд на объект», все подключённые клинеры получают сообщение в Telegram с кнопкой «✅ Взять в работу» — заявку можно забрать прямо из чата.

  1. Создайте бота в @BotFather, получите токен
  2. Укажите его в переменной окружения TELEGRAM_BOT_TOKEN на сервере
  3. Настройте вебхук на адрес __BASE__/telegram/webhook (автоматически, если задать PUBLIC_URL, либо вручную: POST /telegram/setup-webhook?base_url=...)
  4. Каждый клинер/менеджер жмёт «Подключить» в разделе «Менеджеры» напротив своего имени и переходит по ссылке в Telegram
