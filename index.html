Виджет на сайт
Форма заявки для вашего сайта

Добавьте 1 строку кода на сайт — и форма заявки сама отправит данные клиента в CRM.

Вставьте перед </body>:
<script src="https://your-crm-domain.com/widget.js" data-token="nxcrm_UgCTeFkTijVuLD4PFVwx1b-mGYgf7hyemBUfW4BEHcQ" data-pipeline="1"></script>
  1. Замените your-crm-domain.com на адрес вашего сервера
  2. Параметр data-pipeline — ID нужной воронки (1–8)
  3. На сайте появится кнопка «Оставить заявку» с всплывающей формой
  4. Заявки придут в CRM как новые лиды автоматически
REST API
Прямые запросы из любого сервиса

Отправляйте заявки с сайта или из любого сервиса напрямую через HTTP POST-запрос.

Пример curl:
curl -X POST http://localhost:8000/leads \
  -H "Authorization: Basic ТОКЕН" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Имя клиента",
    "phone": "+79991234567",
    "pipeline_id": 1,
    "client_type": "физ"
  }'
  1. Используйте Basic Auth: логин developer, пароль 123456CRM
  2. Поле pipeline_id — номер воронки куда попадёт заявка
  3. Swagger-документация доступна по: /docs
JavaScript / Fetch
Отправка с формы на вашем сайте

Готовый JS-код для вашей существующей формы — скопируйте в обработчик submit.

// В обработчике submit вашей формы:
const creds = btoa('developer:123456CRM');
fetch('http://YOUR-SERVER:8000/leads', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Basic ' + creds
  },
  body: JSON.stringify({
    name: form.name.value,
    phone: form.phone.value,
    pipeline_id: 1,
    client_type: 'физ'
  })
})
.then(r => r.json())
.then(d => console.log('Лид создан:', d));
Telegram-бот
Заявки через Telegram

Создайте Telegram-бота, который будет принимать заявки и автоматически добавлять их в CRM.

  1. Создайте бота через @BotFather в Telegram
  2. Получите токен бота и добавьте Webhook на ваш сервер
  3. В Webhook-хендлере вызывайте POST /leads с данными клиента
  4. Бот будет отвечать клиенту, а заявка — появляться в CRM
Webhook / n8n / Zapier
Автоматизация без кода

Подключите через n8n, Zapier или Make — без кода свяжите CRM с любой формой, Google Sheets, Notion и др.

  1. В n8n создайте HTTP Request-ноду на POST /leads
  2. Укажите Basic Auth: developer / 123456CRM
  3. Маппируйте поля: name, phone, pipeline_id
  4. Тригер — любой: форма, Google Sheet, email и т.д.
Android APK
Собрать приложение за 5 минут

Используйте WebIntoApp.com или AppsGeyser — бесплатно, без кода.

  1. Зайдите на webintoapp.com или appsgeyser.com
  2. Вставьте URL вашего CRM-сервера (напр. https://crm.example.com)
  3. Настройте иконку, название «Nexus CRM», цвет #6D28D9
  4. Нажмите Build — через 2–3 минуты скачайте APK
  5. Установите на Android через «Неизвестные источники»
💡 Для iOS используйте PWA: в Safari откройте сайт → «Добавить на экран» — работает как нативное приложение
